# Run the app locally

This repository is a standalone Vite + React application (no external API key required).

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`

## Tests & Checks

- Unit tests: `npm run test`
- Type-check: `npm run typecheck`
- Production build: `npm run build`
